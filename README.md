# Brown Dust Formation Sharing

## 專案目的

- 建立一個線上編輯陣型的網站
- 提供玩家分享已建立之陣型 
- 不拖泥帶水多餘功能

## 網站功能

- 建立陣型>傭兵選擇>擺放位置>確定>產生圖片或超連結
